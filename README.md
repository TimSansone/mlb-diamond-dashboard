# MLB Diamond Dashboard

A next-generation MLB scores, analytics, highlights, schedules, team pages, AI recaps, and favorite-team notification application.

## Planned features

- Live scores and detailed game centers
- Official video highlight links and broadcast information
- Baseball Savant / Statcast analytics
- Chart.js visualizations
- AI-generated postgame recaps
- Game of the Day ranking
- Complete team pages and monthly calendars
- Light and dark themes
- Server-side caching
- Progressive Web App installation
- Favorite-team push notifications

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment variables

Copy `.env.example` to `.env.local` before enabling database, AI, cache, or notification features.
